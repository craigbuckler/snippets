JavaScript library testbed

The following JavaScript libraries and objects provide cross-browser functionality (IE5.0+, Mozilla, Firefox, Opera, Konqueror and others):

general libraries

  • event handler - define and examine user events
  • miscellaneous functions

DOM manipulation and graphic effects

  • DOM handler - access and manipulate page elements
  • style handler - dynamically switch stylesheets and save user preference
  • popup handler - styles and handles user popup preferences
  • dragdrop handler - drag and drop any page element
  • image scroller - scrolls a large image in the thumbnail area
  • text effects - applies text change effects
  • element effects - collapse or expand any element
  • graphic effects

web service (AJAX) libraries

  • web service handler - consume web services client-side
  • XML handler - load XML and transform using XSLT

form controls

  • form validation handler - easy generic form validation
  • tab control - define tabs and content
  • slider control - define slider controls
  • list control - define reorderable list control
